<h1>Foodie</h1>
<h4>Foodie - The Game, is a simple game where you live every human's dream, 
You play as a mouth and there's food falling down from the sky.</h4>
<h1>Keybinds</h1>
<h4>A -> Move Left</h4><br>
<h4>D -> Move Right</h4><br>
<h4>Esc/P -> Pause</h4><br>
<h4>Esc -> Back</h4><br>
